“How does using a database change how your program manages and protects data compared to saving files?”

By using a database, my information was protected from being modified in that it could only be accessed while using the 
application (also whenever the Visual Studio project was open but imagining this was a real application I am ignoring that),
whereas a text file saved anywhere could be easily opened and modified to suit the modifier's own personal gain.
A database also allows for easy organizing of the data into tables. I can categorize and split the data up so it is
easily accessible via my queries. It can also handle much more information cleanly than a simple text document or csv file. 
I could easily end up with hundreds of entries into my database, and navigating through all of them would be very time consuming unless
the information was kept in a database, where I can easily search for and find the information I need via SQL queries.
